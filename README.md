# Streamflow
### By buidlers, for buidlers.

Streamflow is a suite of payment-related products on Solana.

We're bringing:
- Token vesting
- Stream payments
- Batch (one-to-many) payments
- Multisig payments
- ...and more

To the Solana & Serum ecosystems.
